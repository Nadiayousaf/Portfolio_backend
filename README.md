
# Portfolio Admin Backend (Python)

## 📌 Project Overview

This project is a **Portfolio Admin Backend System** built using **Python** and **JSON** as a database.
It allows the admin to manage portfolio content such as **Skills, Projects, Experience, and Messages** through a command-line dashboard.

This backend will later be connected to a **Frontend Portfolio Website**.

---

## 🚀 Features

### ✅ Admin Authentication

* Secure Admin Login
* Limited login attempts
* JSON-based credential storage

### ✅ Dashboard

* Central Admin Control Panel
* Manage Portfolio Content

### ✅ Skills Management (CRUD)

* Add Skills
* View Skills
* Update Skills
* Delete Skills

### 🔄 In Progress

* Projects Manager

* Frontend Integration
